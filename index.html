Esta página muestra la investigación y ejemplos prácticos sobre las metodologías de nombrado BEM y camelCase utilizando HTML y CSS.

¿Qué es BEM?

BEM (Block Element Modifier) es una metodología que ayuda a escribir código CSS más claro y estructurado.

Ventajas de BEM

  1. Mejor organización del código
  2. Evita conflictos de clases
  3. Facilita el mantenimiento

Ejemplos de clases BEM

  • .menu
  • .menu__item
  • .menu__item--activo

¿Qué es camelCase?

camelCase es un estilo de escritura donde la primera palabra va en minúscula y las siguientes empiezan con mayúscula. Se usa en programación.

¿Dónde se usa camelCase?

  1. Variables en JavaScript
  2. Nombres de funciones
  3. Propiedades en objetos

Ejemplos de camelCase

  • nombreCompleto
  • calcularEdad()
  • miVariable
